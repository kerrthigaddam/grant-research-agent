# grant-research-agent
# Claude-Powered AI Agent for Nonprofit Grant Research & Writing

## Overview
This project helps nonprofits discover funding opportunities and automatically generate grant proposal drafts using AWS Bedrock and Claude.

## Technologies
- Python
- Streamlit
- AWS Bedrock
- Claude Sonnet
- Pandas
- GitHub

## Features
- Grant Matching
- Proposal Generation
- AI-Assisted Writing
- Nonprofit Funding Discovery

## Future Enhancements
- PDF Export
- Bedrock Knowledge Bases
- RAG Search
- Multi-Grant Comparison
